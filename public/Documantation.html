Some help to get started on a new Hoodie project

Ok so first you need to make a new folder, name it what you'd like to do this open the Console first of all.

Then run the command

                  $ mkdir filename

You then want to change into the folders directory with this command

                  $ cd filename
Next you need to create a package.json file by using the command 
                  $ npm init -y
Then we install hoodie 
                  $ npm install hoodie --save
Lastly start it up 
                  $ npm start
And thats it! You now have a working Hoodie app its just kinda empty. If you need more assistence click on this link
